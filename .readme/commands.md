### Команды для работы с проектом:

```
dev - запуск проекта без сборки через ts-node и nodemon

build - линтинг и сборка проекта

start - build и запуск проекта

serv - запуск на сервере в фоновом процессе через pm2

serv:stop - остановка процесса на сервере

lint - линтинг ./src

fix - линтинг с попыткой исправления
```
